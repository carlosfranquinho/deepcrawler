# DeepCrawler

Small browser-based turn-based roguelike with an infinitely deep procedural dungeon.

## Run

- **Option A**: open `index.html` in a browser
- **Option B**: serve the folder (recommended)

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Controls

- **Move**: Arrow keys or WASD
- **Wait**: Space
- **Stairs down**: `>`
- **Stairs up**: `<`
- **Attack**: walk into an enemy
